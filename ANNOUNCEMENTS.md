```
Sefaria MCP Server - Access 5,000 years of Jewish texts from any MCP-compatible LLM
```

### Body
```
Released an MCP server for Sefaria (the open-source library of Jewish texts):

**Tools:**
- `get_text` - Fetch any text by reference (supports Torah, Talmud, Mishnah, Midrash, Halacha, Kabbalah)
- `search` - Full-text search across millions of words in Hebrew/English
- `get_links` - Pull commentaries and cross-references
- `get_calendars` - Daily learning schedules (Daf Yomi, etc.)

**Why this matters for local LLMs:**
Jewish texts have complex reference systems (Berakhot 2a, Genesis Rabbah 1:1, etc.) that LLMs often hallucinate. This gives them grounded, accurate retrieval.

Works with any MCP client - tested with Claude Desktop, Cursor, and the MCP CLI.

**Install:**
```
npx sefaria-mcp-server
```

GitHub: https://github.com/abeperl/sefaria-mcp-server

No API key needed - Sefaria's API is free and open.
```

```
I built an MCP server that connects AI assistants to Sefaria's library of Jewish texts (Torah, Talmud, Mishnah, Midrash, and commentaries).

GitHub: https://github.com/abeperl/sefaria-mcp-server

The Model Context Protocol has 10K+ servers for databases, APIs, and dev tools—but I couldn't find any for religious/historical text corpora. So I built one.

Features:
- Read any text by reference (Genesis 1:1, Berakhot 2a, Rashi on Exodus 20)
- Full-text search in Hebrew or English
- Daily learning calendars (Daf Yomi, weekly Torah portion)
- Cross-references and commentary links

Install: `npm i -g sefaria-mcp-server`

The Sefaria API is free and doesn't require a key. Built with the official MCP SDK.

Use cases I've found useful:
- Research for writing (divrei Torah, academic papers)
- Learning—ask "explain this Talmudic passage" with full context
- Building Torah study apps that need grounded retrieval

Would love feedback on what other text corpora would be useful as MCP servers.
```
